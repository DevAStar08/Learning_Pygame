# Pygame_Basics


Learning Pygame by making mini pygame projects .

Requirements / Pre-requisites :
        Python 2.7 or higher will be fine (I have used 3.7 version).

        Pygame module :
            Install it by just typing the following command in your cmd/terminal -
                     "pip install pygame"
